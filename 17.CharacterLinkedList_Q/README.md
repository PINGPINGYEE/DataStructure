# DataStructure1
과일 리스트를 메뉴로 조작하는 프로그램

- 최초 과일 리스트(10개):  Mango, Orange, Apple, Grape, Cherry, Plum, Guava, Raspberry, Banana, Peach

메뉴

리스트에 새로운 과일 추가

- 입력 받은 과일이 없으면 추가, 있으면 이미 존재한다는 문구 출력
  
(2) 리스트에 있는 과일을 삭제

입력 받은 과일이 있으면 삭제, 없으면 해당 과일이 없다는 문구 출력

(3) 삭제된 과일 목록 출력

- 삭제된 과일 목록은 따로 보관하여 출력
  
(4) 종료
